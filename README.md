# Samsung-Sales-Intelligence and analytics-Dashboard-SQL-Power-BI
Samsung sales intelligence dashboard using sql and power BI

## Project Overview
This project presents an end-to-end data analysis workflow using SQL and Power BI to derive insights from Samsung sales data.

## Dataset
- ~15,500 Sales records
- Includes product, pricing, region , and transaction data.

Tools Used
- SQL
- Power BI
- DAX

## Key Metrics
-Total Revenue: $22.9M
-Total Orders: 16K
- Return Rate: ~14%
- Total Units Sold: 36K+
 
## Key Insights
- Premium products generate the highest revenue
- Strong regional performance in Europe & Middle East
- Return rate (~14%) indicates product gaps
